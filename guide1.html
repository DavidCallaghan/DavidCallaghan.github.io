QGIS to Leaflet to Github Pages page

    Qgis
      Get all your data displayed how you want it
        Note: everything might have to be geoJSONs, but not sure
      Add the plugin “QGIS2Web” to your QGIS In the plugin menu select make a map Fiddle around with the options Press export The folder structure and web page will appear in whatever folder you selected in the options in the plugin
      Github Pages
        Go to https://pages.github.com/ Follow the instructions to create a “Repository” (the bit of Github where you are going to host your page with the map on it), on Github online Download Github Desktop Open your new repository in Github Desktop by clicking the button that says “Setup in desktop”
        Your PC or Mac’s file explorer
          Copy the complete folder structure and HTML file for the webmap into the local folder that holds the repository for the Github page
          Github Desktop
            Github Desktop should detect that you’ve made changes to the repository Bottom left click “Commit” It’ll do some thinking, then take you to another page in the app that has a button that says “Push”, click that and wait till it does its stuff
              Access webmap online
                In a browser go to https://WHATEVER-YOUR-GITHUB-ACCOUNT-NAME-IS.github.io/THE-FILENAME-OF-THE-HTML-PAGE-YOU-EXPORTED-FROM-QGIS.html
